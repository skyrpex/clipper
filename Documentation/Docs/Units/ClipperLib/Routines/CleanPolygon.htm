CleanPolygon

Del.» function CleanPolygon(const Poly: TPolygon; Distance: double = 1.415): TPolygon;

C++ » void CleanPolygon(Polygon &in_poly, Polygon &out_poly, double distance = 1.415);

C#  » public static Polygon CleanPolygon(Polygon poly, double distance = 1.415);


Removes:

  • vertices that join co-linear edges
  • a vertex that's within the specified distance of an adjacent vertex
  • a vertex that's within the specified distance of a semi-adjacent vertex together with the out-lying vertex

The default distance approximates √2 so that if vertices have their corresponding X and Y coordinates differ by no more than 1 unit, then one of the vertices will be removed (plus the out-lying vertex if they are semi-adjacent vertices).
